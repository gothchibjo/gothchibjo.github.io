# Follow-Us MVP

Простой прототип сервиса follow-up по итогам встречи.

## Что уже есть
- Форма создания follow-up документа.
- Автоформирование итогового текста в заданном шаблоне.
- Сохранение документов в `localStorage`.
- Открытие сохраненных документов и по hash-ссылке `#/doc/<id>`.
- Автоподстановка участников и тем из предыдущих встреч.
- Отчет по задачам: срез по срокам и ответственным.
- Экспорт в `md`, `docx`, `pdf`.

## Запуск
Рекомендуемый запуск через локальный сервер:

```bash
python3 -m http.server 8080
```

После этого откройте:
- [http://localhost:8080/index.html](http://localhost:8080/index.html)

Запуск через `file://` может блокировать часть скриптов браузером (CORS/Module restrictions).

## Ограничение текущего прототипа
Хранение локальное, поэтому "ссылка на документ" работает в рамках того же браузера/профиля, где документ сохранен.
