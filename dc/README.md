# Азбука узора: Цифровой код нации
## Интерактивная карта орнаментов народов России

Веб-пакет для раздела сайта. Проект «Компьютерное Зазеркалье», 19 сезон.

---

### Структура файлов

```
ornament-map/
├── index.html          ← главная страница
├── css/
│   └── map.css         ← все стили
├── js/
│   ├── map.js          ← логика карты (ES-модуль)
│   ├── patterns.js     ← SVG-паттерны орнаментов + данные округов
│   └── map-data.js     ← SVG-содержимое карты России (269 KB)
└── README.md
```

---

### Как разместить на сайте

**Вариант 1 — статический хостинг (GitHub Pages, Netlify, Vercel)**

Загрузите папку `ornament-map/` целиком. Откройте `index.html`.
Работает без сборщиков и бэкенда — только статические файлы.

> ⚠️ ES-модули (`type="module"`) требуют HTTP/HTTPS.
> Напрямую через `file://` не откроется — нужен локальный сервер.

**Вариант 2 — встроить в существующий сайт**

Скопируйте папки `css/` и `js/` в нужный раздел сайта.
В вашем HTML подключите:
```html
<link rel="stylesheet" href="путь/к/css/map.css">
```
И перед `</body>`:
```html
<script type="module">
  import { MAP_SVG_INNER } from './js/map-data.js';
  import { initMap }        from './js/map.js';
  initMap(MAP_SVG_INNER);
</script>
```

**Вариант 3 — локальный просмотр**

```bash
# Python 3
python3 -m http.server 8080
# затем открыть http://localhost:8080/ornament-map/
```

### Функциональность

| Действие | Результат |
|---|---|
| Навести на регион | Орнамент плавно заполняет округ (400 мс) |
| Убрать курсор | Орнамент плавно исчезает (300 мс) |
| Нажать «Показать все орнаменты» | Все округа заполняются с анимацией (stagger 60 мс) |
| Нажать повторно | Все орнаменты исчезают |
| Чекбокс «Все орнаменты сразу» | Синхронизирован с кнопкой |
| Навести на легенду | Активирует соответствующий округ |

### Требования к браузеру

- ES Modules (Chrome 61+, Firefox 60+, Safari 10.1+, Edge 16+)

- CSS Custom Properties

Не требует jQuery, React или других зависимостей.
